# 🚀 Constraint-Uncertain Linear Programming (CULP)

> Bridging Natural Language and Optimization using Large Language Models

---

## 📌 Overview

This repository presents a prototype implementation of **Constraint-Uncertain Linear Programs (CULPs)** — integrating:
- Natural Language Processing (LLMs)
- Operations Research (Linear Programming)
- Soft + Hard Constraints

---

## 🧩 Workflow

Natural Language → Constraint Extraction → Classification → CULP Model → Solver → Explainable Output

---

## 🏥 Example: Hospital Scheduling

- Ensure minimum staffing
- Limit workload
- Allow flexible preferences

```
pip install -r requirements.txt
```
