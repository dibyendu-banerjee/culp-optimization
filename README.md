
# 🚀 Constraint-Uncertain Linear Programming (CULP)

> Bridging Natural Language and Optimization using Large Language Models

---

## 📌 Overview

This repository presents a prototype implementation of **Constraint-Uncertain Linear Programs (CULPs)** — a novel optimization framework that integrates:

- 🧠 Natural Language Processing (LLMs)
- 📊 Chance-Constrained Optimization
- ⚙️ Operations Research (Linear Programming)

The goal is to transform **human intent** into **mathematically rigorous optimization models**.

---

## 🎯 Key Idea

Traditional optimization requires rigid constraints.

CULP introduces:
- 🔴 **Hard Constraints** → Must be satisfied  
- 🟡 **Soft Constraints** → Satisfied probabilistically  

This enables **flexible, real-world decision-making**.
